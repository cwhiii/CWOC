# Release 20260504.1838

Added true Obsidian-style Live Preview to the Notes zone. The render button in the zone header now cycles through three modes: Source (raw markdown textarea), Live Preview (rendered markdown per-block that reveals raw syntax when you click into a block — exactly like Obsidian), and Reading (fully rendered, non-editable). A formatting toolbar with buttons and keyboard shortcuts (Ctrl+B, Ctrl+I, Ctrl+K, Ctrl+E, Ctrl+Shift+X/1/2/3/7/8/./-) is shown in Source and Live Preview modes. The email body zone retains its existing inline preview behavior.
