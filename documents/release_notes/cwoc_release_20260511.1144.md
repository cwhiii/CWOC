Overhauled notification lifecycle. Notifications now persist forever — dismiss changes status to "dismissed" (not delete). Notifications view split into two sections: Unread (pending, top) and Addressed (dismissed/accepted/declined, bottom), both sorted most-recent-first. Each card has a permanent delete button (🗑️). "Clear Addressed" button bulk-deletes the addressed section. Desktop-only reminders auto-dismiss after first toast so they don't re-fire on subsequent page loads. Backend PATCH now accepts "dismissed" status.
