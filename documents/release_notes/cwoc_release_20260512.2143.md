Added Custom Zones — users can now create named collections of Custom Objects that render as additional collapsible zones in the chit editor. Zones are managed from the Custom Objects Editor (create, rename, reorder, delete, assign/remove objects with drag-to-reorder) and automatically appear on all chits with the same field rendering, range highlighting, conditional display, and unit system support as the built-in Health Indicators zone. Data is stored in the existing UUID-keyed health_data field on each chit.
