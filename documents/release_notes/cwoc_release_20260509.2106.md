# Release 20260509.2106

Map thumbnails on chit cards: Tasks, Checklists, and Assigned-to-Me views show a small OpenStreetMap tile thumbnail for non-default locations. Notes view uses just the map-marker icon in the header instead. Calendar, Alarms, and Projects show a compact icon. Double-clicking any map thumbnail opens the Maps page centered on that location. Per-user toggle in Settings → Chit Options.
