Replaced "Default Filters" in Settings with "Custom Filters & Sorting" — per-view buttons that open a modal with the full sidebar filter UI (status, priority, display toggles, text search, sort). Saved custom filters auto-apply when switching to that view on the dashboard. The "Clear" button on the dashboard now resets to user-defined custom defaults (or system defaults if none set). Supports all 9 views (Omni first, then tab order). Backward-compatible with legacy default_filters and omni_locked_filters.
