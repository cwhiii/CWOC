Added desktop-only notification delivery. Notifications in the editor alerts zone now have "next time on desktop" in the unit dropdown. On save, a deferred notification is created that only surfaces on non-mobile screens. When it arrives, it fires both an in-app toast and a browser system notification. Profile menu no longer shows individual notification cards — replaced with a single "X notifications pending" banner at the bottom that navigates to Alarms→Notifications view when clicked. All notification surfaces handle reminder-type with Dismiss (no Accept/Decline).
