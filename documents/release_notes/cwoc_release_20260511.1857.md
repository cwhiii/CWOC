Added prerequisites feature to the Task zone. Chits can now depend on other chits — when any prerequisite is incomplete, the chit is auto-blocked; when all are complete, it resets to ToDo. The picker uses a new shared `cwocChitPickerModal()` utility (same table/search/filter UI as "Add Child Chits" in Projects) with multi-select checkboxes. Prerequisites display as full-color items with auto-contrast text and inline-editable status dropdowns. Task zone dropdowns are now uniform width with aligned left edges. Also includes circular dependency detection, manual override with warning, cascade unblocking, and ⛓️ chain indicator on cards.
