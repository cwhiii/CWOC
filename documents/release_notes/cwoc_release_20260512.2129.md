Added weather-based notifications as a direction option in the notification dropdown (alongside Desktop, At, Before, After). When a chit has a location, "Weather Condition" appears as an option with sub-controls for: temperature thresholds (high/low above/below), precipitation type (rain, snow, hail), or wind gusts over a speed. Backend weather fetch now includes wind gust data. Hint text updated to guide users toward adding a time or location for more notification options.
