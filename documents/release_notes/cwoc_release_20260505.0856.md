# Release 20260505.0856

- Mobile: Added touch drag support for checklist items in the editor (hold to grab, drag to reorder)
- Added undo/redo buttons (↩/↪) to checklist, notes, and email zone headers
- Checklist undo/redo also works via Cmd+Z / Cmd+Shift+Z when the checklist input is focused
- Added "📝→☑️" button to copy note lines into checklist items (respects indentation and list markers)
- Added "☑️→📝" button to copy checklist items into the note field (with indentation and check state)
