Added prerequisites feature to the Task zone. Chits can now depend on other chits — when any prerequisite is incomplete, the chit is auto-blocked; when all are complete, it resets to ToDo. Uses the same dropdown pattern as "Add to Project" with checkmarks for multi-select. Prerequisites display as full-color items (using the chit's color as background with auto-contrast text), with inline-editable status dropdowns. Task zone dropdowns are now uniform width with aligned left edges. Also includes circular dependency detection, manual override with warning, cascade unblocking, and ⛓️ chain indicator on cards.
