Added desktop-only notification delivery. Notifications in the editor alerts zone now have "next time on desktop" in the unit dropdown. On save, a deferred notification is created that only surfaces on non-mobile screens. When it arrives (page load or poll), it fires both an in-app toast and a browser system notification. The profile menu, sidebar inbox, and Alarms→Notifications view all handle reminder-type notifications with a Dismiss button (no Accept/Decline). All notification fetch calls now pass device type for proper filtering.
