Added desktop-only notification delivery. In the editor alerts zone, the notification unit dropdown now includes "next time on desktop" — selecting it hides the number/timing inputs and creates a deferred notification that only surfaces when the app is opened on a non-mobile screen. Shows in both the sidebar notification inbox (red dot + entry) and the Notifications mode of the Alarms view. Backend filters by device type via new `delivery_target` column and `?device=` query param.
