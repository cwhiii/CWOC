Added prerequisites feature to the Task zone. Chits can now depend on other chits — when any prerequisite is incomplete, the chit is auto-blocked; when all are complete, it resets to ToDo. Uses the same dropdown pattern as "Add to Project" but with checkmarks for multi-select. Includes circular dependency detection, manual override with warning, cascade unblocking when prerequisites complete, and a ⛓️ chain indicator on cards with incomplete prerequisites.
