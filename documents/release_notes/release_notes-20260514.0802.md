Tag filter bar: clicking a tag toggles its checkbox, Shift+Click selects only that tag (deselecting all others). Added "Any Tag" and "Tagless" virtual options at the top of the tag filter — "Any Tag" is selected by default (showing all chits regardless of tags), "Tagless" shows only untagged chits. Selecting a real tag disables both virtual options. If nothing is selected, no chits appear.
