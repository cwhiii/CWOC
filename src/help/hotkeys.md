CWOC provides extensive keyboard shortcuts for fast navigation and actions. Press `?` on the dashboard to see the full hotkey overlay.

## Dashboard Navigation

- `O` — [Omni View](/frontend/html/help.html#omni-view)
- `S` — Sort/Order submenu
- `G` — [Global Search](/frontend/html/help.html#global-search)
- `L` — Clocks modal
- `W` — [Weather page](/frontend/html/weather.html) (Shift+W for quick modal)
- `V` — Navigate panel
- `.` — Period selector
- `?` — Hotkey reference overlay
- `F4` — Calculator
- `F9` — Settings
- `Shift+R` — Help page

## View Switching

Use the C CAPTN tab bar or hotkeys to switch between Calendar, Checklists, Alerts, Projects, Tasks, and [Notes view](/frontend/html/help.html#notes)s.

## Editor Shortcuts

- `Ctrl+S` / `Cmd+S` — Save
- `Escape` — Exit (with save check)
- `Ctrl+B` / `Cmd+B` — Bold (in notes/email/checklist items)
- `Ctrl+I` / `Cmd+I` — Italic (in notes/email/checklist items)
- `Ctrl+K` / `Cmd+K` — Link (in notes/email/checklist items)
- `Ctrl+E` / `Cmd+E` — Inline code (in notes/email/checklist items)
- `Ctrl+Shift+X` / `Cmd+Shift+X` — Strikethrough (in notes/email/checklist items)

### Checklist-Specific

- `Enter` — Split item at cursor, create new item below
- `Shift+Enter` — Insert newline within item
- `Tab` / `Shift+Tab` — Indent / outdent item
- `Cmd+[` / `Cmd+]` — Indent / outdent (single item)
- `Cmd+Shift+(` / `Cmd+Shift+)` — Indent / outdent item + children
- `Arrow Up/Down` — Navigate between items
- `Ctrl+Click` / `Cmd+Click` — Toggle multi-select on item
- `Shift+Click` — Range select items
