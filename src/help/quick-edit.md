# Quick Edit Modal

The Quick Edit modal provides fast inline editing without leaving the dashboard. Open it by Shift+clicking a chit card (or long-pressing ~1.2s on mobile).

The modal shows the chit title, status, priority, dates, tags, and notes in a compact form. Make changes and they save automatically when you close the modal.
