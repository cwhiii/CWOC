# Mouse Interactions

- **Click** a chit card — Open in the editor
- **Shift+click** a chit card — Open [Quick Edit modal](/frontend/html/help.html#quick-edit)
- **Middle-click** a chit card — Open in a new browser tab
- **Double-click** a calendar event — Open in the editor
- **Drag** a calendar event — Move to a new time/day
- **Drag edge** of a calendar event — Resize (change duration)
- **Short hold (~0.4s) + drag** in [Notes view](/frontend/html/help.html#notes) — Reorder notes between columns
- **Long-press (~1.2s)** on mobile — Open [Quick Edit modal](/frontend/html/help.html#quick-edit)
