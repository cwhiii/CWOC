# Calculator

Press `F4` on any page to open a floating calculator popover. Press the same hotkey again (or `ESC`) to close it. The calculator provides basic arithmetic with correct operator precedence and can insert results directly into editor fields.

## Arithmetic

The calculator supports addition (+), subtraction (−), multiplication (×), and division (÷) with standard operator precedence — multiplication and division are evaluated before addition and subtraction. Press `=` or `Enter` to evaluate. Press `C` to clear, or `⌫` / `Backspace` to delete the last character. Division by zero shows an error.

## Keyboard Input

While the calculator has focus, type digits (0–9), operators (+, -, *, /), press `Enter` for equals, `Backspace` to delete the last character, and `ESC` to close. Keyboard input is captured by the calculator and does not propagate to page hotkeys.

## Insert Result

Click **Insert Result** to write the current calculator result into the field you were editing when you opened the calculator. This only works on the [chit editor](/editor) page — the button is disabled on all other pages and when no source field was focused.

## Draggable

Drag the calculator's title bar to reposition it anywhere on screen. The calculator stays within the visible viewport. Works with both mouse and touch.

## ESC to Close

Pressing `ESC` while the calculator is open closes it before any other ESC action (closing modals, exiting the editor, etc.). When the calculator is not open, ESC behaves normally.
