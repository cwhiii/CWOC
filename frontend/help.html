What is CWOC?

C.W.'s Omni Chits is a personal task, note, and calendar management app. The core concept is a chit — a flexible record that can be a task, note, calendar event, alarm, checklist, or project, all in one data model.

Chits

A chit is the universal unit. What makes it appear in different views is which fields you fill in:

  • Calendar — has a start/end date or due date
  • Tasks — has a status (ToDo, In Progress, Blocked, Complete)
  • Notes — has note content
  • Checklists — has checklist items
  • Alerts — has alarms, timers, or notifications
  • Projects — marked as a "Project Master" with child chits

A single chit can appear in multiple views if it has the relevant fields.

Views & Tabs

The top tab bar switches between six views: Calendar, Checklists, Alerts, Projects, Tasks, Notes. The sidebar (toggle by clicking the logo) provides sorting, period selection, and filters.

Calendar View

Periods: Itinerary, Day, Week, Work Hours, X Days, Month, Year. Switch with the Period dropdown or . hotkey.

  • Start & end times → timed blocks
  • Due date only → ⌚ icon
  • All-day → all-day bar at top
  • Work Hours → only configured working hours and days
  • X Days → configurable count (Settings → Period Options)

Mouse Interactions

On Calendar Events

  • Double-click — Open in editor
  • Shift+click — Quick Edit modal
  • Drag up/down — Move time · Drag left/right — Move day
  • Drag bottom edge — Resize (start/end events only)

On Empty Space

  • Double-click — Create new chit at that time

Notes View

  • Double-click — Open in editor
  • Shift+click — Edit note in place

Quick Edit Modal

  • Double-click title — Edit title in place
  • 📱 QR — Data QR (Shift+click for link QR)

Keyboard Shortcuts

  • K — Create chit (middle-click → new tab)
  • . — Period: I D Week K Work S X-Day Month Year
  • F — Filter: Status Tags Priority D Filter Text
  • O — Order: Title Start Due Updated Created X Status Manual Random G Upcoming
  • G — Global Search
  • W — Weather (default location forecast)
  • L — Clock · R — Reference · Shift+R — Help · S — Settings
  • ESC — Close/exit · Backspace — Clear filters

Editor Hotkeys

Recurrence / Repeating

Check 🔁 Repeat in the Dates zone. Patterns: Daily, Weekly, Monthly, Yearly, Custom. Optional end date.

Instance Actions (Shift+click)

  • ✏️🔁 Edit series · ✏️1️⃣ Edit instance · ✏️✂️ Break off & edit

Drag prompts: This instance only / All in series / All following.

Tags

Hierarchical labels with colors. Nesting via / (e.g., Work/Projects/CWOC). Child tags inherit parent color. ★ Favorites appear at top.

Global Search

Press G or click the magnifying glass tab (last in the C CAPTN tab bar) to open the Global Search view. Unlike the sidebar filter, Global Search queries ALL fields across ALL chits regardless of which tab they belong to.

  • Search Input + Go — Type a query in the search input and press Enter or click Go. The search performs case-insensitive substring matching across every chit field (title, note, tags, status, priority, location, people, checklist items, dates, color, alerts, and more).
  • Result Cards — Each matching chit is displayed as a card showing the chit title and every matched field with the matching text highlighted. Click a result card to open that chit in the editor.
  • Sidebar Filters — Sidebar filters (Status, Priority, Tags, etc.) remain active during Global Search, narrowing results further.

Filtering & Sorting

Sidebar: Filter Text, Status, Priority, Tags, Show (Pinned/Archived/Unmarked/Hide Past-Due). 💾 Save Search saves text as a quick-access chip.

Notes View

Multi-column masonry layout with full rendered markdown. [[chit title]] auto-links to other chits.

Shift+click (or long-press on mobile) a note card to edit it in place. While editing, drag-and-drop reordering is disabled so you can click to position the cursor. Press ESC or click outside to save and exit edit mode.

Clocks

Press L on the dashboard to open the clocks modal. Configure which clocks are displayed and their layout in Settings → Alarms.

  • 24 Hour — Standard 24-hour digital display (HH:MM:SS)
  • 12 Hour — 12-hour digital display with AM/PM
  • 12 Hour Analog — Traditional analog clock face with hour, minute, and second hands
  • HST — Holeman Simplified Time. Represents the day as 0–100 sub-days (sd) shown on a progress bar. Midnight = 0.000 sd, noon = 50.000 sd, end of day = 100.000 sd

Drag clocks between Active and Inactive zones in Settings to choose which appear. Use "Change Orientation" to switch between horizontal and vertical layout.

Weather

Press W on the dashboard to open the weather modal. It shows the current forecast for your default saved location, including weather icon, high/low temperatures (°F), and precipitation. If no default location is configured, the modal will prompt you to set one in Settings.

In the chit editor, weather loads automatically when a chit has both a location and a date. Placeholder messages appear when either is missing.

On the dashboard, chits with a location display a weather icon (☀️ sunny, 🌧️ rainy, 🌨️ snowy, etc.) next to the title. Hover over the icon to see high/low temperatures and expected precipitation. Weather data is fetched in the background and cached for one hour.

Saved Locations

Manage saved locations in Settings → Saved Locations. Each location has a label and an address. Use the radio button to mark one as the default.

  • Adding — Click ➕ Add Location to add a new row. Fill in a label and address.
  • Default — Select the radio button next to a location to make it the default. If only one location has an address, it is auto-selected.
  • Removing — Click ✕ to remove a row. At least one row always remains.
  • Saving — Empty-address rows are cleaned up on save.

Saved locations are used by the editor's Location zone dropdown, the compact 📍 dropdown in the title area, the +Location button, and the dashboard weather modal (W).

Chit Editor

Collapsible zones: Title, Dates & Times, Task, Location, Tags, People, Notes, Checklist, Alerts, Health Indicators, Color, Projects, Audit Log. 📱 QR button generates data or link QR codes.

When creating a new chit, only the zone relevant to the current view is expanded (e.g., Notes zone when coming from the Notes tab). All other zones start collapsed. The title field is auto-focused. If you have a default saved location, it's auto-applied to new chits.

Alerts Zone

Add alarms, timers, stopwatches, and notifications. Each alarm has a time, days, and optional "Delete chit after dismissal" checkbox — when checked, dismissing the alarm soft-deletes the chit. The 🔄 button on the weather bar refreshes the forecast.

Task Zone

Status (ToDo, In Progress, Blocked, Complete), Priority (High, Medium, Low), and Severity (Critical, Major, Normal, Minor).

Location Zone

  • +Location — Populates the address from your default saved location
  • Saved Locations dropdown — Select any saved location by label to populate the address
  • ✕ Clear — Clears the address, map, and weather display
  • 📍 compact dropdown — Quick-access dropdown in the title/weather area for selecting a saved location without scrolling to the Location zone

A chit can have at most one location. Weather and map load automatically when a location and date are both present.

Quick Edit Modal

Shift+click any chit. Edit title (double-click), change Priority/Severity/Status, Pin/Archive/Delete/QR. Recurring chits get per-field "All" checkboxes and series edit options.

Settings

  • General — Time format (24h, 12h, HST), snooze length, calendar snapping
  • Period Options — Enable/disable periods, week start, working hours & days, X Days count
  • Default Filters — Per-tab default search text
  • Clocks — Choose active clocks, drag to reorder, set orientation (horizontal/vertical)
  • Tag Editor — Tree view, create/edit/delete/favorite, "+" for child tags
  • Saved Locations — Add, label, and set a default location for use across the editor and dashboard weather
  • Custom Colors — Color picker for chit colors
  • Chit Options — Fade past, highlight overdue, delete past alarms
  • Export / Import — CSV export and import
  • Trash — View, restore, or permanently delete chits

Alert Indicators

Alert indicators are small icons displayed alongside chit titles to signal that a chit has one or more alerts (alarms, notifications, timers, or stopwatches). Configure their visibility in Settings → Visual Indicators.

Combine Alerts Toggle

The Combine Alerts checkbox in Visual Indicators switches between two display modes:

  • Combined mode (checked) — A single ⚠️ icon represents all alert types. A "Combined Alerts" dropdown controls visibility (Always / Never / If Space).
  • Individual mode (unchecked) — Separate icons appear per alert type, each with its own visibility dropdown: 🔔 Alarm, 📢 Notification, ⏱️ Timer, ⏲️ Stopwatch.

Timer & Stopwatch Dropdowns

When Combine Alerts is off, two new dropdowns appear alongside the existing Alarm and Notification controls:

  • Timer — Controls visibility of the ⏱️ timer indicator (Always / Never / If Space)
  • Stopwatch — Controls visibility of the ⏲️ stopwatch indicator (Always / Never / If Space)

Both default to "Always" if not previously configured.

Calendar Views

Calendar views (Week, Day, Month, Year, Itinerary, Seven-Day, X-Day) always show a single ⚠️ icon regardless of the Combine Alerts toggle, since calendar cells are space-constrained.

Card Views

Checklists, Tasks, Notes, and Alarms views respect the Combine Alerts toggle — showing either one combined ⚠️ icon or individual per-type icons depending on the setting. The Projects view does not show alert indicators.

"If Space" Behavior

When a visibility dropdown is set to "If Space," indicators always appear on card views and calendar day/week slots (sufficient room). On month cells, indicators may be hidden when the title would be truncated.

Version Management

The Settings page includes a Version & Updates section that shows the currently installed version number and the date/time it was last updated.

  • Version Info — Displays the current version and "Last Updated" timestamp. If no version info is available, a fallback message is shown.
  • Upgrade Omni Chits — Click this button to trigger an update. A modal opens showing real-time output from the update script.
  • Update Modal — Streams the update log in a terminal-style display. Log lines are color-coded by prefix: [OK] green, [WARN] yellow, [ERROR] red, [STEP] blue. The Close button is enabled once the update finishes.

Contact Editor

Contacts have name fields (prefix, given, middle, surname, suffix), phone, email, address, social/web, security (Signal, PGP), and context (organization, social context).

  • Notes — Markdown notes field for free-form text about the contact
  • Tags — Tags for contacts, auto-prefixed with "Contact/" (shared with the general tag system)
  • Suffixes — Text suffixes (Jr., Sr., Esq., Ph.D., M.D.) listed before numeric suffixes (I–X)
