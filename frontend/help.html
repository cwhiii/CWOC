What is CWOC?

C.W.'s Omni Chits is a personal task, note, and calendar management app. The core concept is a chit — a flexible record that can be a task, note, calendar event, alarm, checklist, or project, all in one data model.

Chits

A chit is the universal unit. Every item you create is a chit. What makes it appear in different views is which fields you fill in:

  • Calendar — has a start/end date or due date
  • Tasks — has a status (ToDo, In Progress, Blocked, Complete)
  • Notes — has note content and no dates
  • Checklists — has checklist items
  • Alerts — has alarms, timers, or notifications
  • Projects — marked as a "Project Master" with child chits

A single chit can appear in multiple views if it has the relevant fields.

Views & Tabs

The top tab bar switches between the six views: Calendar, Checklists, Alerts, Projects, Tasks, Notes. Each filters chits by the relevant criteria.

The sidebar (toggle by clicking the logo) provides sorting, period selection (for calendar), and filters.

Calendar View

Calendar has multiple periods: Itinerary, Day, Week, Work Hours, 7-Day, Month, Year. Switch with the Period dropdown or the . hotkey.

  • Chits with start & end times appear as timed blocks
  • Chits with only a due date appear with a ⌚ icon
  • All-day events appear in the all-day bar at the top
  • The current time is shown as a dark brown line
  • Today is highlighted in all calendar views

Mouse Interactions

On Calendar Events

  • Double-click — Open the chit in the full editor
  • Shift+click — Open the Quick Edit modal (change status, priority, pin, archive, delete, and recurrence options)
  • Drag up/down — Move the event to a different time
  • Drag left/right — Move to a different day (Week/7-Day views)
  • Drag bottom edge — Resize (change end time). Only for start/end events, not due-only.

On Empty Calendar Space

  • Double-click — Create a new chit at that time slot. The start time is pre-populated from where you clicked, with a 1-hour default duration.

Keyboard Shortcuts

Press R to see the full reference overlay. Key shortcuts:

  • C H A P T N — Switch tabs
  • K — Create new chit (middle-click opens in new tab)
  • . — Period submenu: Itinerary, Day, weeK, Work Hours, S 7-Day, Month, Year
  • F — Filter submenu → Status, Tags, Priority, Words
  • O — Order/sort submenu (G for Upcoming, R for Random)
  • L — Clock modal
  • R — Reference overlay
  • Shift+R — This help page
  • S — Settings
  • Backspace — Clear the active filter (or all filters from the F menu)
  • ESC — Close/exit current submenu

Editor Hotkeys

  • Alt+1 — 📅 Dates & Times
  • Alt+2 — 📋 Task (Status / Priority / Severity)
  • Alt+3 — 🏷️ Tags
  • Alt+4 — 📝 Notes
  • Alt+5 — ✅ Checklist
  • Alt+6 — 🔔 Alerts
  • Alt+7 — 📍 Location
  • Alt+8 — 👥 People
  • Alt+9 — 🌈 Color
  • Alt+0 — 📊 Projects
  • ESC — Exit editor
  • Type [[ in Notes to auto-complete and link to other chits

Recurrence / Repeating

Any chit with dates can repeat. In the editor's Dates & Times zone, select a recurrence pattern:

  • Daily, Weekly, Monthly, Yearly — simple presets
  • Custom — "Every N [minutes/hours/days/weeks/months/years]" with optional day-of-week checkboxes for weekly
  • Ends — optional end date for the series

Working with Recurring Instances

On the calendar, each occurrence appears as a virtual instance with a 🔁 icon. Shift+click opens the Quick Edit modal with recurrence-specific options:

  • Edit series — opens the parent chit in the editor (changes affect all instances)
  • Edit this instance — opens the editor for just this date (saved as an exception)
  • Break off & edit — creates a standalone copy of this instance and removes it from the series
  • Skip this instance — hides this occurrence without deleting the series
  • Complete this instance — marks just this date as done
  • Complete entire series — marks the parent chit as Complete

Dragging Recurring Events

When you drag a recurring instance to a new time, a prompt asks:

  • This instance only — moves just this occurrence (saved as an exception)
  • All in series — changes the time for all instances
  • All following — changes the time for this and future instances

Series Info

The Quick Edit modal shows the instance number (#3 of series), completion stats (e.g., "5/12 completed, 42% success rate"), and a collapsible Series Summary listing all instances with their status.

When a series has an end date and all instances are completed or skipped, the parent chit is automatically archived.

Tags

Tags are hierarchical labels with colors. Create and manage them in Settings. Tags support nesting with / separators (e.g., Work/Projects/CWOC).

  • Tags appear with their color as background everywhere (editor, sidebar, views)
  • Favorite tags (★ gold star) appear at the top of filter lists and in the editor's favorites row
  • Filtering by a parent tag also matches all child tags

Filtering & Sorting

The sidebar has collapsible filter sections:

  • Search — text search across titles and notes
  • Status — multi-select (ToDo, In Progress, Blocked, Complete). "Any" means no filter.
  • Priority — multi-select (Low, Medium, High)
  • Tags — tree view with checkboxes, search box
  • Show — toggle Pinned, Archived, Unmarked visibility

Each filter section has a Clear button. Press Backspace in a filter hotkey submenu to clear that filter.

Notes View

Notes display in a multi-column masonry layout (like Google Keep). Cards are assigned to columns and maintain their position when reordered.

  • Drag & drop to reorder within or between columns
  • ESC during drag cancels and restores original position
  • Markdown is rendered in note cards
  • Double-click a card to open in the editor

Chit Editor

The editor has collapsible zones:

  • Title — with pin toggle and recurrence icon
  • Dates & Times — radio buttons for None/Due/Start&End, All Day checkbox in the header, recurrence picker
  • Task — Status, Priority, Severity dropdowns. The Due line has a Complete checkbox that syncs with Status.
  • Location — address input with map search and directions
  • Tags — tree view, favorites, recent, active tags panel
  • People — comma-separated names
  • Notes — markdown editor with render toggle, expand modal, copy, download
  • Checklist — nested items with drag-drop, indent/unindent
  • Alerts — alarms, timers, stopwatches, notifications
  • Health Indicators — (stub, not yet functional)
  • Color — swatch picker with custom colors from settings
  • Projects — Kanban board for project master chits

Save buttons: Save & Stay (keep editing), Save & Exit (return to views), Exit (discard if unsaved).

Validation: if a date mode is active, a date and time (or All Day) are required before saving.

Quick Edit Modal

Shift+click any chit on the calendar to open the Quick Edit modal. It provides:

  • Task fields — editable dropdowns for Priority, Severity, Status (only shown if the chit already has values). For recurring chits, changes apply to this instance only by default. Check "Apply to all instances" to change the entire series.
  • Pin / Archive / Delete — toggle buttons in a row. Delete shows sub-options for recurring chits (this instance, this and following, all).
  • Recurrence options — (only for recurring chits) edit series, edit instance, break off & edit
  • Series summary — collapsible list of all instances with completion status
  • Open in editor — full editor access

Settings

Accessible via the sidebar or S hotkey:

  • Time Format — 24-hour, 12-hour, or metric
  • Calendar Snapping — snap interval for drag operations (5-60 min)
  • Tag Editor — create, edit, delete, favorite tags with colors. Displayed as a tree view.
  • Custom Colors — add colors for the chit color picker
  • Snooze Length — default snooze duration for alarms

Save buttons: Save & Stay, Save & Exit, Cancel.
