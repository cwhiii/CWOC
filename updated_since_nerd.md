# CWOC Features Added Since April 28, 10 PM

**Email client** — full IMAP/SMTP with compose, reply, threading, attachments, search, signatures, formatting toolbar

**Rules engine** — "if this then that" automation with condition trees, 16+ actions, visual builder, scheduled rules

**Habits system** — goal tracking, period rollover, charts, streaks, success rates, reset periods, perpetual mode

**Maps view** — Leaflet map with chit/contact markers, clustering, filtering, geocode caching

**Sharing & collaboration**
- Sharing overhaul — invite/assign, manager permissions, notification inbox, tag-level sharing
- RSVP system — accept/decline on shared chits
- Contact Vault — share contacts server-wide with read-only access

**Mobile & connectivity**
- PWA wrapper — installable app with offline support and push notifications
- Ntfy push notifications — phone notifications with action buttons, timer scheduling, per-user topics
- Tailscale integration — VPN config with status monitoring and connect/disconnect
- Mobile touch drag — tap-and-hold reorder across all views with floating cards

**People & contacts**
- People zone overhaul — full-screen modal, sortable table, role toggles, unified contact/user editor
- Contact dates — birthday/anniversary fields with vCard support

**Editor improvements**
- Checklist overhaul — item counts, collapsible done section, Clear Checked with undo
- Calculator popover — F4 hotkey arithmetic calculator
- iCal import — import .ics files from Google/Apple/Outlook calendars

**Kiosk view** — tag-based public display with period navigation
