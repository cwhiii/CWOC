# Calendar View

Periods: Itinerary, Day, Week, Work Hours, X Days, Month, Year. Switch with the Period dropdown or `.` hotkey.

- **Start & end times** → timed blocks
- **Due date only** → ⌚ icon
- **All-day** → all-day bar at top
- **Point-in-time** → outward-pointing diamond shape (◇)
- **Birthday/Anniversary** → concave-notch shape (inward cutouts on left and right edges), with contact thumbnail and 🎂 emoji. Click to open the contact editor. Searchable by name or date label
- **Work Hours** → only configured working hours and days
- **X Days** → configurable count (Settings → Period Options)
